## Environments

| Package    | Version |
| :--------- | :------ |
| Node       | 18.13.0 |
| React      | 18.2.0  |
| Typescript | 4.9.5   |

<br/>

## How to install

you can download modules with command:

```
yarn install
```

and you can start with command:

```
yarn start
```

<br/>
